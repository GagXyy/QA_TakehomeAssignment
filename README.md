# QA Take-Home Assignment

## Overview

This repository contains my implementation of the QA Take-Home Assignment. The solution is organized into multiple projects using Java and Maven.

The assignment requirements have been completed as follows:

* **Task 1 – Manual Test Cases:** Included in the `uiTests` project.
* **Task 2 – UI Automation:** Implemented in the `uiTests` project using Selenium WebDriver and TestNG.
* **Task 3 – API Automation:** Implemented in the `api-tests` project using Rest Assured and TestNG.

Additionally, I have included an extra project demonstrating the same UI automation scenarios using the **Cucumber BDD** framework.

---

## Task 1 – Manual Test Cases

Location: **uiTests**

Includes manually designed test cases covering the OrangeHRM Login functionality.

## Task 2 – UI Automation

Location: **uiTests**

### Technology Stack

* Java 18
* Selenium WebDriver
* TestNG
* Maven
* Page Object Model (POM)

### Automated Test Cases

* Verify login with valid username and password
* Verify user can log out successfully
* Verify login with invalid username
